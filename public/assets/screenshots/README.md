# Localized Landing Screenshots

Put localized screenshots here. The landing page reads these files based on `?locale=`:

- `dashboard_en.png`
- `training_en.png`
- `exercises_en.png`
- `muscles_en.png`
- `dashboard_ua.png` (optional)
- `training_ua.png`
- `exercises_ua.png`
- `muscles_ua.png`
- `dashboard_ru.png`
- `training_ru.png`
- `exercises_ru.png`
- `muscles_ru.png`

If a locale-specific image is missing, the site falls back to:

- `assets/account.management/screen_home.png`
- `assets/account.management/screen_profile.png`
